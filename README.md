# E-commerce Frontend

This is a modern e-commerce frontend application built with React, TypeScript, and Vite. It features product listings, shopping cart functionality, user authentication, and more.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
